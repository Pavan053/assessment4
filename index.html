What is Css?

  1. Css stand for Cascadding style sheeet.
  2. Css discribe how to HTML element are to be displayed on screen ,paper,or in other media.
  3. Sss is not neccesary Css saves a lot of work.It can control the layout of multiple web page.
  4. External stylesheet are stored in CSS files.

Css can be added to HTML document in 3 way:

  • Inline- by using the style attribute inside the HTML element .
  • Internal-by using a style element in the head section .
  • External-by using a link element to link an xternal CSS files.
  • The most common way to add CSS ,is to keep the style in external CSS files. However , in this tutorial we will use inline and internal style ,because this is easier to demondtrate ,and easier for you to try it yourself.
